# paintjs
[Painting Board made with Vanilla JS](https://nomadcoders.co/javascript-for-beginners-2/lobby)

### 기능 요소

- 색칠 기능
  - Paint - 그림 그리기
    - 브러쉬 크기 조절 가능
  - Fill - 화면 채우기
- 저장 기능
  - Save 버튼을 통해 .png 파일로 저장 기능
- 초기화 기능
  - Init 버튼을 통해 화면 초기화 기능
